## Portfolio site – Vadim Korolkov

This is a simple static implementation of the portfolio site inspired by `vadim-korolkov-en.figma.site`.

### Structure
- `index.html` – main single-page portfolio
- `styles.css` – layout and visual design
- `script.js` – small interactions (if needed later)

### How to run
Just open `index.html` in any modern browser.
